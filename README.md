# Image Gallery Project

A modern and responsive image gallery built using **HTML, CSS, and JavaScript**.

This project was created to practice frontend development concepts such as layouts, animations, DOM manipulation, responsive design, and interactive UI components. The gallery is designed with smooth transitions, hover effects, 3D styling, and a clean user experience.

## Features

- Beautiful gallery layout using HTML and CSS
- Modern UI with glassmorphism and 3D hover effects
- Responsive design for mobile, tablet, and desktop
- Smooth animations and transitions
- Lightbox preview for viewing images in large size
- Next / Previous navigation buttons
- Category-based image filtering
- Hover effects for better user interaction
- Clean and organized code structure

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## How It Works

### Gallery Layout
Images are displayed in a responsive grid layout. CSS Grid and Flexbox are used to make sure the gallery adjusts properly on different screen sizes.

### Hover Effects
Each image card includes:
- Scale animation
- Shadow effects
- Smooth transitions
- 3D tilt-like feel

These effects improve the visual appeal and create a more interactive experience.

### Lightbox View
When a user clicks an image:
- A popup opens
- Selected image appears in large view
- User can navigate using:
  - Next button
  - Previous button
  - Close button

This makes browsing easier without leaving the gallery page.

### Filters and Categories
Images can be filtered by categories such as:
- Nature
- Animals
- Travel
- Technology

Users can quickly switch between categories to view relevant images.

## Responsive Design

The gallery is fully responsive and tested for:

- Mobile devices
- Tablets
- Laptops
- Large screens

Media queries are used to optimize spacing, image sizing, and layout.
